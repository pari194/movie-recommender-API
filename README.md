Movie Recommender API :

A movie recommendation system built using FastAPI and collaborative filtering based on the MovieLens dataset.

Features :
1. Recommend similar movies using user ratings
2. REST API built with FastAPI
3. Dataset from MovieLens
4. Cosine similarity based collaborative filtering

Tech Stack :
Python  
FastAPI  
Pandas  
Scikit-learn  

Run the project :

1. Clone the repository
2. Install dependencies
